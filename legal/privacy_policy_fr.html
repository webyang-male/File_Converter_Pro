Politique de Confidentialité de File Converter Pro

Dernière mise à jour : Février 2026

1. Introduction

Nous prenons très au sérieux la protection de vos données personnelles. Cette politique de confidentialité explique comment File Converter Pro traite vos informations.

2. Philosophie de Confidentialité

File Converter Pro est une application 100% hors ligne. Nous croyons fermement que vos fichiers vous appartiennent et doivent rester sur votre machine.

3. Collecte des Informations

3.1 Données Personnelles

Nous ne collectons AUCUNE donnée personnelle. Aucune information vous concernant n'est envoyée à nos serveurs ou à des tiers.

3.2 Traitement des Fichiers

Tous les fichiers que vous convertissez, fusionnez, compressez ou modifiez sont traités localement sur votre ordinateur. Aucun fichier n'est jamais envoyé sur Internet.

4. Utilisation des Fichiers

Vos fichiers sont :

  • ✅ Traités uniquement sur votre machine - Aucun transfert vers des serveurs externes
  • ✅ Non envoyés à des tiers - Nous ne partageons jamais vos fichiers
  • ✅ Supprimés selon vos instructions - Vous contrôlez complètement vos données
  • ✅ Stockés temporairement si nécessaire - Seulement pour les opérations en cours

5. Données Techniques Locales

5.1 Fichiers de Configuration

L'application sauvegarde localement vos préférences (thème, langue, paramètres) dans un fichier de configuration chiffré sur votre machine.

5.2 Historique des Conversions

Un historique local de vos conversions est stocké sur votre machine pour votre convenance. Vous pouvez le supprimer à tout moment via les paramètres.

5.3 Statistiques d'Utilisation

Certaines statistiques anonymes peuvent être collectées localement (nombre de conversions, formats utilisés) pour améliorer l'expérience utilisateur. Ces données restent sur votre machine.

6. Partage d'Informations

Aucune information n'est partagée avec des tiers. Nous ne vendons pas, ne louons pas et ne partageons pas vos données avec aucune entreprise ou organisation.

7. Sécurité

Nous mettons en œuvre des mesures de sécurité robustes pour protéger vos fichiers :

  • 🔒 Chiffrement local des fichiers de configuration sensibles
  • 🔒 Isolation des processus pour éviter les fuites de données
  • 🔒 Nettoyage automatique des fichiers temporaires après utilisation
  • 🔒 Permissions minimales - L'application n'accède qu'aux fichiers que vous sélectionnez

8. Cookies et Technologies Similaires

L'application n'utilise pas de cookies. Étant une application de bureau hors ligne, aucun cookie ou technologie de suivi web n'est utilisé.

9. Vos Droits

En tant qu'utilisateur, vous avez le droit de :

  • 👁️ Accéder à toutes les données stockées localement
  • ✏️ Modifier vos préférences à tout moment
  • 🗑️ Supprimer complètement toutes les données de l'application
  • ⚙️ Exporter vos paramètres et historique

10. Consentement

En utilisant File Converter Pro, vous consentez à notre politique de confidentialité et reconnaissez que :

  • ✅ Tous les traitements sont effectués localement
  • ✅ Aucune donnée n'est collectée ou envoyée
  • ✅ Vous êtes responsable de la sauvegarde de vos fichiers

11. Modifications de la Politique

Nous pouvons mettre à jour cette politique de confidentialité. Toute modification importante sera notifiée lors du lancement de l'application.
