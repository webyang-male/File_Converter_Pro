Informativa sulla Privacy di File Converter Pro

Ultimo aggiornamento: Febbraio 2026

1. Introduzione

Prendiamo molto sul serio la protezione dei vostri dati personali. Questa informativa sulla privacy spiega come File Converter Pro gestisce le vostre informazioni.

2. Filosofia della Privacy

File Converter Pro è un'applicazione 100% offline. Crediamo fermamente che i vostri file vi appartengano e debbano rimanere sul vostro computer.

3. Raccolta delle Informazioni

3.1 Dati Personali

Non raccogliamo ALCUN dato personale. Nessuna informazione che vi riguarda viene inviata ai nostri server o a terze parti.

3.2 Elaborazione dei File

Tutti i file che convertite, unite, comprimete o modificate vengono elaborati localmente sul vostro computer. Nessun file viene mai inviato su Internet.

4. Utilizzo dei File

I vostri file sono:

  • ✅ Elaborati solo sul vostro computer - Nessun trasferimento verso server esterni
  • ✅ Non inviati a terze parti - Non condividiamo mai i vostri file
  • ✅ Eliminati secondo le vostre istruzioni - Avete il pieno controllo sui vostri dati
  • ✅ Memorizzati temporaneamente se necessario - Solo per le operazioni in corso

5. Dati Tecnici Locali

5.1 File di Configurazione

L'applicazione salva localmente le vostre preferenze (tema, lingua, impostazioni) in un file di configurazione cifrato sul vostro computer.

5.2 Cronologia delle Conversioni

Una cronologia locale delle vostre conversioni è memorizzata sul vostro computer per vostra comodità. Potete eliminarla in qualsiasi momento tramite le impostazioni.

5.3 Statistiche di Utilizzo

Alcune statistiche anonime possono essere raccolte localmente (numero di conversioni, formati utilizzati) per migliorare l'esperienza utente. Questi dati rimangono sul vostro computer.

6. Condivisione delle Informazioni

Nessuna informazione viene condivisa con terze parti. Non vendiamo, affittiamo o condividiamo i vostri dati con alcuna azienda o organizzazione.

7. Sicurezza

Implementiamo misure di sicurezza robuste per proteggere i vostri file:

  • 🔒 Cifratura locale dei file di configurazione sensibili
  • 🔒 Isolamento dei processi per prevenire perdite di dati
  • 🔒 Pulizia automatica dei file temporanei dopo l'uso
  • 🔒 Permessi minimi - L'applicazione accede solo ai file che selezionate

8. Cookie e Tecnologie Simili

L'applicazione non utilizza cookie. Essendo un'applicazione desktop offline, non vengono utilizzati cookie né tecnologie di tracciamento web.

9. I Vostri Diritti

In qualità di utente, avete il diritto di:

  • 👁️ Accedere a tutti i dati memorizzati localmente
  • ✏️ Modificare le vostre preferenze in qualsiasi momento
  • 🗑️ Eliminare completamente tutti i dati dell'applicazione
  • ⚙️ Esportare le vostre impostazioni e la cronologia

10. Consenso

Utilizzando File Converter Pro, acconsentite alla nostra informativa sulla privacy e riconoscete che:

  • ✅ Tutta l'elaborazione viene eseguita localmente
  • ✅ Nessun dato viene raccolto o inviato
  • ✅ Siete responsabili del backup dei vostri file
